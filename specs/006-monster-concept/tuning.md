# Monsters & Opponents Tuning Guide

## Time-to-Kill (TTK) Targets (Solo Player)
- **Minion (Swarm)**: < 1s
- **Minion (Bruiser)**: 2–4s
- **Elite**: 8–15s
- **Champion**: 45–90s (assuming coordination)
- **Boss**: 3–5 minutes

## Break Thresholds (Impact Units)
- **Champion**: 2–3 heavy impacts (e.g., 100-150 points)
- **Boss**: 5–8 heavy impacts (e.g., 250-400 points)

## Recovery Windows
- **Minion**: 0.2s
- **Elite**: 0.5s–1.0s
- **Champion**: 1.5s–2.0s
- **Boss**: Explicit phase transitions / 3s windows
