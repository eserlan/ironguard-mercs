# Progression Tuning Guide

## XP Curves (Run Progression)
- **Base Threshold**: 1000 XP
- **Growth Factor**: 1.2x per level
- **Max Run Level**: 20 (Soft cap)

## Power Budgets
- **Common Perk**: ~2–5% power nudge
- **Rare Augment**: ~10–15% skill-specific amp
- **Legendary Trait**: Systemic behavior change (e.g., "Full heal on wave clear")

## Micro Slow Targets
- **Duration**: 8.0s
- **Slow Factor**: 0.25 (75% slow)
- **Cooldown**: 30s per player
